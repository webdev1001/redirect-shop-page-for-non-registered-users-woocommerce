== Description ==

It Redirects Wocommerce shop page, to my account page for non registered users which is customizable, So that users can't see these pages without register/login.


Major features in Woocommerce Redirect include:

* Admin panel option to enable non registered users to redirect to my-account-page for shop,cart page




== Installation ==

Upload the this plugin to your blog, then go to settings and click Woocommerce Redirect, select all settings and click save options .

1, 2, 3: You're done!
